本地运行
-------------------------

安装 [node.js](http://nodejs.org/)

然后用npm安装项目所依赖的包

```javascript
sudo npm install
```
安装bower
```javascript
npm install bower -g
```
安装rsvp依赖包
```javascript
bower install
```
启动node
```javascript
node server.js
```
访问页面
```javascript
http://localhost:8080/app/html/phone.html
```
